✅ YOUR PROJECT IS 95% COMPLETE!

Just 3 simple steps to get it running:

───────────────────────────────────────────────────────────────
STEP 1: GET YOUR FREE CREDENTIALS (5 minutes)
───────────────────────────────────────────────────────────────

📦 MongoDB Atlas (FREE Forever):
   1. Visit: https://www.mongodb.com/atlas/database
   2. Click "Try Free" → Sign up
   3. Create a FREE cluster (M0 Sandbox - 512MB)
   4. Click "Connect" → "Connect your application"
   5. Copy the connection string

🖼️  Cloudinary (FREE 25GB):
   1. Visit: https://cloudinary.com/users/register/free
   2. Sign up for free account
   3. Go to Dashboard
   4. Copy: Cloud Name, API Key, API Secret

───────────────────────────────────────────────────────────────
STEP 2: UPDATE ENVIRONMENT VARIABLES
───────────────────────────────────────────────────────────────

Edit the file: .env.local

Replace these values with your real credentials:

MONGODB_URI=mongodb+srv://YOUR_USER:YOUR_PASS@cluster0.xxxxx.mongodb.net/portfolio
NEXT_PUBLIC_CLOUDINARY_CLOUD_NAME=your_actual_cloud_name
CLOUDINARY_API_KEY=your_actual_key
CLOUDINARY_API_SECRET=your_actual_secret

───────────────────────────────────────────────────────────────
STEP 3: RUN YOUR PORTFOLIO
───────────────────────────────────────────────────────────────

In your terminal, run:

   npm run dev

Then open your browser to:

   http://localhost:3000

🎉 That's it! Your portfolio is now running!

🆘 TROUBLESHOOTING:
───────────────────────────────────────────────────────────────

Problem: Can't connect to MongoDB
Solution: 
   - Whitelist your IP in MongoDB Atlas
   - Check username/password in .env.local
   - Ensure you replaced <username> and <password>

Problem: Port 3000 already in use
Solution: 
   lsof -ti:3000 | xargs kill -9
   npm run dev

Problem: Cloudinary upload fails
Solution:
   - Verify credentials in .env.local
   - Check file size (must be < 10MB)
   - Ensure it's a valid image file
